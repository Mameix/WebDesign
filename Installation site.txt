Créer un virtual Host :
	- Copier le dossier WebDesign dans www de Wamp
	- Démarrer Wamp
 	- Aller sur localhost/
	- Cliquer sur ajouter un Virtual Host :
		Nom host : master-miage
		Chemin absolu : C:\wamp64\www\WebDesign\Site

Accès au site : 
	- Aller sur https://master-miage
	- Voir identifiants connexion dans le fichier correspondant
